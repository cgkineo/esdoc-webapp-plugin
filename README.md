# ESDoc WebApp Plugin
Makes ESDoc function as though it were a single page web app.
This helps with page transition smoothness and keeping the scroll position.

## Install
```bash
npm install esdoc-webapp-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc-webapp-plugin"}
  ]
}
```
